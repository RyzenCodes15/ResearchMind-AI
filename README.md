# ResearchMind AI

ResearchMind AI is a production-grade AI Research Assistant built with Retrieval-Augmented Generation (RAG).

## Features (Planned)

- Secure user authentication
- Upload and process research papers (PDF)
- Semantic search using embeddings
- AI-powered document chat
- Paper summaries
- Flashcard generation
- Quiz generation
- Paper comparison
- Conversation history
- Modern responsive UI

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy 2.x
- PostgreSQL

### AI
- LangChain
- Ollama
